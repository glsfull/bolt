# bolt

Репозиторий содержит исходный прототип сервиса "Мои анализы", перенесенный из
Bolt.new, и техническое задание на его переработку в полноценный продукт.

## Что находится в репозитории

- `project/` - текущий React/Vite-прототип. Он используется как эталон дизайна,
  экранов, текстов, блоков и пользовательских сценариев.
- `project/screenshot_*.html` - сохраненные HTML-эталоны ключевых экранов.
- `docs/mobile-rewrite-technical-spec.md` - техническое задание на перенос в
  десктопную web-версию, мобильную web-версию и мобильное приложение.

## Цель переработки

Новая реализация должна сохранить визуальную и функциональную структуру текущего
прототипа: дизайн, стили, расположение кнопок и блоков, тексты, навигацию,
таблицы, карточки и основные сценарии пользователя. При этом production-версия
должна получить полноценные frontend/backend-слои, защищенное хранение
медицинских данных, Supabase-интеграции, OCR/AI-обработку на backend-стороне и
отдельную административную зону.

Основной документ для оценки и следующей реализации:

[Техническое задание и план переработки](docs/mobile-rewrite-technical-spec.md)
